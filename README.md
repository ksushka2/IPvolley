# Лабораторная работа №3: "Мой первый хоумпейдж"

## Описание проекта

Веб-сайт о волейболе, созданный в рамках лабораторной работы. Сайт состоит из трёх связанных страниц с информацией о волейболе, его истории и правилах игры.

## Структура проекта

```
├── index.html      # Главная страница
├── history.html    # Страница об истории волейбола
├── rules.html      # Страница с правилами игры
├── styles.css      # Файл стилей CSS
├── scripts.js      # JavaScript файл с интерактивными функциями
└── README.md       # Этот файл
```

## Требования и реализация

### ✅ Реализованные элементы HTML:

1. **Мета-теги** - присутствуют на всех страницах (charset, viewport, description, keywords, author)
2. **Заголовки разных уровней** - используются h1, h2, h3
3. **Тексты** - информационный контент на всех страницах
4. **Изображения с обтеканием текстом** - используется float-left и float-right
5. **Таблицы** - таблицы с данными на всех страницах
6. **Списки** - маркированные (ul) и нумерованные (ol) списки
7. **Внутренние ссылки (якоря)** - навигация по странице через #id
8. **Внешние ссылки** - ссылки на внешние ресурсы (открываются в новой вкладке)

### ✅ Реализованные JavaScript скрипты:

1. **Удаление слов более 7 букв** - функция `removeLongWords()`
   - Результат выводится в pop-up окне (alert)

2. **Поиск наиболее часто встречающейся буквы** - функция `findMostFrequentLetter()`
   - Результат выводится в pop-up окне (alert)

3. **Удаление цифр из текста** - функция `removeDigits()`
   - Результат выводится в новом окне (window.open)

4. **Изменение цвета фона** - функция `changeBackgroundColor()`
   - Кнопки для изменения цвета фона страницы

5. **Автоматическая прокрутка текста** - функция `startAutoScroll()`
   - Прокрутка начинается через 5 секунд после нажатия кнопки

6. **Часики в заголовке (Title)** - функция `updateTitleClock()`
   - Время обновляется каждую секунду в заголовке страницы

7. **Вывод текущей даты** - функция `showCurrentDate()`
   - Выводит текущую дату и время в pop-up окне

## Как использовать

### Локальный просмотр:

1. Откройте файл `index.html` в браузере
2. Используйте навигационное меню для перехода между страницами
3. Протестируйте все JavaScript функции на главной странице

### Загрузка на хостинг:

1. Выберите бесплатный хостинг (например, GitHub Pages, Netlify, Vercel, 000webhost)
2. Зарегистрируйте доменное или субдоменное имя
3. Загрузите все файлы проекта на сервер через FTP/SFTP/SSH или Git
4. Убедитесь, что файлы находятся в корневой директории сайта
